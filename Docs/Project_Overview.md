# Project Overview

## Introduction

This project demonstrates the implementation of a complete DevSecOps CI/CD pipeline for deploying a static web application using modern cloud-native technologies.

The pipeline automates code analysis, security scanning, containerization, image publishing, and deployment to a Kubernetes cluster.

## Objectives

- Automate application delivery
- Implement DevSecOps practices
- Perform static code analysis
- Perform vulnerability scanning
- Containerize applications
- Deploy applications on Kubernetes
- Enable automated deployments through Jenkins

## Technologies Used

- GitHub
- Jenkins
- SonarQube
- Trivy
- Docker
- Docker Hub
- Kubernetes
